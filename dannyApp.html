Documents

This is a documents module. This module will display a list of documents associated with this corpus, such as scans of field notes or archival documents, copies of consent forms, etc. Clicking a document will display the document details in the details pane.

    Here is a <ul> with id=documentsList for displaying the list of documents associated with this corpus. It may be necssary to have multiple <ul>’s for different types of documents.

Lexicon

This is a lexicon module. This module will display the list of entries in the lexicon or lexica associated with this corpus, and can be filtered and sorted, etc. Clicking a lexical entry will display that entry in the details pane.

    Here is a <ul> with id=lexiconEntries for displaying the list of lexical entries for this corpus.

Media

    Orthographies

    This is an orthographies module. This module will display the list of orthographies associated with this corpus. Clicking an orthography will display the orthography in the details pane for editing.

      Here is a <ul> with id=orthographiesList for displaying the list of orthographies associated with this corpus. It may be necessary to have multiple <ul>’s for different languages.

    Tags

    This is a tags module. This module will display a list of tags in the corpus, organized in some way. It may be that this is removed later and integrated into other types of tools, like a corpus tagger. Alternatively, this may be the place where those tools live.

      Here is a <ul> with id=tagsList for displaying the list of tags associated with this corpus.

    Texts

      Details Pane

      This is the details pane, where the user will be doing most of their work. Users will use the overview pane to pull up items to work on, and then input and edit those items here.

      Each of the modules in this section will be hidden unless the user has selected the relevant view from the navigation menu at the top of the page.

      This is a documents module. Details of individual documents - or perhaps the document itself - will be displayed here.

      This is a lexicon module. Details of individual lexical entries will be displayed here.

      This is an orthographies module. Details of individual orthographies will be displayed here.

      This is a tags module. Details of individual tags will be displayed here (for example, all the phrases a tag has been applied to).
